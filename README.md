## 👉 0408 회원 서비스 프로젝트
- 모든 입력은 스캐너(콘솔 입력) 클래스를 이용합니다.
- 필수 회원 정보는 id, pw, 이름 입니다. (여유가 있다면 유효성 검사 추가)
- DB를 대체할 클래스 DAO (스프링에서 Repository)는 별도로 구현합니다.
  - class DAO는 필드로 Map을 선언하여, DB 대체로 활용합니다. <br>
    (=DB의 기능은 Map의 메소드가 대체) <br>
